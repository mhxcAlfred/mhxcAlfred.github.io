力扣00067.二进制求和

发表于 2020年03月24日 第13周 第084天 星期二 分类于 语言 阅读次数：
二进制求和、位运算、数学、字符串、模拟、简单

题目描述

给你两个二进制字符串 a 和 b ，以二进制字符串的形式返回它们的和。

示例 1：

输入:a = “11”, b = “1”
输出：”100”

示例 2：

输入：a = “1010”, b = “1011”
输出：”10101”

提示：

  • $1 <= a.length, b.length <= 10^4$
  • a 和 b 仅由字符 ‘0’ 或 ‘1’ 组成
  • 字符串如果不是 “0” ，就不含前导零
